Pokanalysis

Pokanalysis analyses first generation Pokémons ROM… At least Red and Blue… US. ☺

It's a totally useless application (unless you are just curious about the game) since you can't edit the ROM with it. I insist on it: Pokanalysis is a VIEWER, not an editor.

Also, you won't find any ROM here.

Dependencies

  • python >= 2.5
  • PyGTK
  • cmake (build only)
  • gcc (build only)

How to build & use

% tar -zxf pokanalysis-*.tar.gz
% cd pokanalysis-*
% mkdir build
% cd build
% cmake ..
% cd ..
% make -C build
% ./pokanalysis /path/to/your/pokemon/gb/rom.gb
